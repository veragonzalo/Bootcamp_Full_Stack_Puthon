Mi página CSS + FLEXBOX

Aplicando estilos, tipografía, cajas y flexbox.

¿Qué estoy practicando hoy?

En esta página estoy aplicando los conceptos básicos de CSS: tipografía, márgenes, padding y un toque de Flexbox.

  • Uso de una hoja de estilos externa.
  • Mejorar la legibilidad del texto.
  • Organizar el contenido en secciones claras.
  • Dar un estilo más moderno usando el modelo de cajas.

Pasos que seguí

  1. Crear la estructura básica de HTML5.
  2. Conectar el archivo estilos.css desde el <head>.
  3. Ajustar tipografía, tamaños y espaciados.
  4. Centrar el contenido principal usando márgenes automáticos y aplicar detalles visuales como bordes redondeados y sombras suaves.
