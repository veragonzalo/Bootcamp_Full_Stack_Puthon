Gestión de tareas en una empresa

Contexto:

En una empresa de desarrollo de software, los equipos deben gestionar sus tareas diarias de manera eficiente. Para ello, necesitan una herramienta sencilla que les permita agregar y eliminar tareas en una lista dinámica.

El área de gestión ha solicitado una funcionalidad que permita a los empleados agregar tareas pendientes y eliminarlas cuando estén completadas, sin necesidad de recargar la página.

Instrucciones:

Desarrollar una aplicación web simple en la que los empleados puedan agregar tareas a una lista y eliminarlas una vez que estén completadas, utilizando jQuery para manipular el DOM.

Aplicación de ejemplo

Tip: haz clic en una tarea para marcarla como completada y eliminarla de la lista.
