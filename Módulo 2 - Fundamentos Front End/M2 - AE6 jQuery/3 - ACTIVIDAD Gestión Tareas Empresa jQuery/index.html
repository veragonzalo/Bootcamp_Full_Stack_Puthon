Gestión de tareas en una empresa

Contexto:

En este ejercicio, crearemos una aplicación web interactiva utilizando jQuery para gestionar usuarios. Los administradores podrán agregar, actualizar y eliminar usuarios dinámicamente en una tabla HTML, aplicando eventos de jQuery para manipular el DOM.

Instrucciones

  • Crear una interfaz con una tabla para mostrar los usuarios registrados.
  • Agregar un formulario para ingresar nuevos usuarios con nombre y edad.
  • Permitir editar los datos de un usuario directamente desde la tabla.
  • Implementar la opción de eliminar usuarios mediante un botón en cada fila.
  • Usar jQuery para manejar los eventos y actualizar la interfaz en tiempo real, sin recargar la página.
